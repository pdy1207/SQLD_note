#### **NULLIF** === 특정 값을 NULL로 대체하는 경우

`NVL` (표현식1 , 표현식2) : 표현식1의 결과값이 NULL이면 표현식 2의 값을 출력한다.<BR>
`NULLIF` (표현식1, 표현식2) : 표현식1 이 표현식2 와 같으면 NULL을, 같지 않으면 표현식 1을 리턴<BR>
`COALESCE` (표현식1, 표현식2) : 임의의 개수 표현식에서 NULL이 아닌 최초의 표현식을 나타낸다. <BR>
